Correlation

Correlation is a Statistical analysis taht measures the strength and direction of the relationship between two or more variables .

Correlation is of two types : 1 . Positive Correlation , 2 . Negative Correlation

Now we can know what is positive and negative coorelations .

1 . Positive Correlation : In Machine Learning , when two variables move in the same direction , meaning that as one variable increases so that the other value is at constant .Then we say it as Positive Correlation .

2 . Negative Correlation : It is the relationship between the two variables that move in opposite directions , when one variable increases so that the other decreases and vice versa .

Train_Test_Split Method

train_test_split method is a technique that divides a Dataset into two subsets one is a training set & a test set . And is generally used in Supervised Machine Learning for Regression & Classification of model .

1 . Train set : Training set is used to train the model .

2 . Test set : The Test set is used to evaluate the model's performance .

While we are training and testing the sets it contains the text_size ( describes the number that defines the size of the test set i.e., here we taken as 25%(0.25)) and random_state ( is a parameter that controls the random number generator used to shuffle the data before splitting it ) within the paranthesis .
